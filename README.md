# Airbnb analysis for Turing

This repository is the work I've done for analyzing airbnb with respect to a data
analysis assignment for Turing.

## Usage

```
$ python -m venv .
$ source bin/activate
$ pip install -r requirements.txt
$ python download_data.py
$ jupyter notebook
```
